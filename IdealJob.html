My Ideal Job

My ideal position involves working in a Network Operations Centre as this provides a great mix of fast paced work, problem solving and exposure to different hardware-software mixes.

Ideal position: IP NOC Engineer

Seek job link: https://bit.ly/2W6X0CY

This Network Operations Centre (NOC) Engineer role involves network monitoring and traffic management, fault rectification and network provisioning throughout the entire TPG IP network. The inherent requirement of this position is an ability to problem solve in a fast-paced environment, work to timeframes and liaise with vendors and other NOC staff; tasks which I find exciting. Tasks include responding to network faults and trouble tickets, assisting in planned outages and network upgrades, using network monitoring tools to maintain oversight of various elements of the TPG IP network, and working with hardware and technology vendors in relation to network faults, performance and monitoring.
